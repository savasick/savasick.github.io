Sava About

Linux

Linux is a free and open-source operating system kernel that powers various devices. It is known for stability, security, and flexibility.

Bash

Bash is a Unix shell and command language. Bash offers a powerful scripting language and advanced features, making it a popular choice for automation and system interaction.

Docker

Docker is an open-source platform for building, deploying, and running applications in containers lightweight, portable, and self-contained environments.

Git

Git is a distributed version control system for tracking changes in source code during software development, enabling collaboration and code management.

PHP

PHP is a server-side scripting language for web development, known for its flexibility and ease of use.

Python

Python is a versatile, high-level language known for simplicity and extensive libraries, making it popular for web development, data analysis, and more.

Java Script

JavaScript is a high-level programming language for creating interactive elements on websites and web applications.

HTML

HTML is the standard markup language for creating and structuring web pages.

CSS

CSS is a styling language that controls the look and layout of web pages.

Django

Django is a high-level Python web framework that encourages rapid development and clean design.

Laravel

Laravel is a PHP web application framework known for its expressive syntax and structured approach to building web apps.

Vue

Vue.js is a JavaScript framework for building interactive web interfaces. It provides benefits of reactive data binding and composable view components with an API.

PostgreSQL

PostgreSQL is a powerful, open source object-relational database system. It uses and extends SQL, providing many features for safely storing and scaling complex data workloads.

MySQL

MySQL is a free, open-source relational database management system. It is widely used for managing and storing data in web applications.

NMAP

Nmap is a free and open-source network scanning and discovery tool. It is widely used by security professionals and network administrators to map networks, detect live hosts, identify open ports, and uncover running services and operating systems.

Wireshark

Wireshark is a network protocol analyzer that captures and displays network traffic at a microscopic level. It is widely used for network troubleshooting, security analysis, and education.
